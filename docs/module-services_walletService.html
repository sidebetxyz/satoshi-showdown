Module: services/walletService

Service for managing wallets in Satoshi Showdown. This service provides essential functionalities for creating, retrieving, updating, and managing cryptocurrency wallets, particularly Segregated Witness (SegWit) Bitcoin wallets. These wallets are primarily used for handling financial transactions related to events. The service interfaces with the wallet data model and external key generation utilities to ensure secure and efficient management of wallet-related operations.

Creates a SegWit Bitcoin wallet specifically for handling event-related financial transactions.
