Module: routes/eventRoutes

Routes for Event Management in Satoshi Showdown. This module defines the Express routes for handling event-related operations such as creating, updating, deleting, and retrieving events. Each route is associated with a specific controller function to handle the request. These routes are integral to the application's RESTful API, allowing clients to interact with event data stored in the backend system.
