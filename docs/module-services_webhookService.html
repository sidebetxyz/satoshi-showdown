Module: services/webhookService

Service for managing webhooks in Satoshi Showdown. This service is responsible for creating, updating, retrieving, and managing the deletion of webhooks. It interfaces with external services like BlockCypher for blockchain event monitoring and interacts with the local database for webhook data management. The service plays a crucial role in event-driven processes within the application, especially for transaction and blockchain event monitoring.
